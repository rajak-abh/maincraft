# MainCrafts Landing Page

A modern, responsive landing page for **MainCrafts**, a digital product and technology services brand. The page is built as a static website using HTML, CSS, and vanilla JavaScript.

## Overview

This website presents MainCrafts as a company that builds fast, scalable, and secure digital products. It includes a polished hero section, service navigation, feature cards, animated statistics, a contact/application form, and a footer with company links.

## Features

- Fixed desktop navigation with a services dropdown
- Mobile hamburger menu
- Full-screen hero section with call-to-action buttons
- Responsive feature cards
- Animated scroll reveal effects
- Smooth statistics counter animation
- Contact/application form layout
- Footer with service links, company links, contact details, and social icons
- Responsive design for desktop, tablet, and mobile screens

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts
- Font Awesome Icons

## How to Run

No installation is required because this is a static website.

1. Open the project folder.
2. Double-click `maincrafts_landing.html`.
3. The website will open in your default browser.

You can also open it using a code editor extension such as Live Server for automatic refresh while editing.

## Website Sections

### Hero

Introduces the MainCrafts brand with the headline **Build Smarter with MainCrafts**, supporting copy, and two action buttons: apply now and view features.

### Features

Highlights the main strengths of the service:

- Lightning Fast
- Fully Responsive
- Infinitely Scalable
- Secure by Default

### Statistics

Displays animated company metrics such as projects delivered, client satisfaction, team members, and years in the industry.

### Contact

Provides a simple form for users to send a message or apply for an internship position.

### Footer

Includes the brand summary, social links, service links, company links, and contact information.

## Customization

To update the page content, edit `maincrafts_landing.html`.

Common changes include:

- Updating the company name, tagline, or hero text
- Changing contact email and website links
- Editing service names in the dropdown and footer
- Updating feature card text
- Replacing statistics values in the HTML and JavaScript
- Adjusting colors in the CSS variables inside the `<style>` block

## Notes

The page uses external CDN links for Google Fonts and Font Awesome, so an internet connection is required for fonts and icons to load correctly.
